# IT Security Policy
**Policy ID:** IT-SEC-001
**Version:** 1.0
**Effective Date:** 2026-01-01
**Policy Owner:** Information Security

## 1. Purpose
This policy defines acceptable technology use, company device rules, personal device restrictions, password security, public Wi-Fi guidance, and incident reporting requirements.

## 2. Company Device Usage
Employees must use company-managed devices for business work whenever possible.

Company devices must have:
- endpoint protection enabled
- disk encryption enabled
- automatic security updates
- screen lock enabled
- approved monitoring and management tools

## 3. Personal Laptop Restrictions
Employees should not use personal laptops for regular office work.

Personal laptop use is allowed only when:
- there is a temporary business need
- manager approval is obtained
- IT security approval is obtained
- no confidential or customer data is stored locally
- the device meets minimum security requirements

The use of personal laptops for customer data, production access, or confidential project documents requires explicit IT security approval.

## 4. Password Security
Employees must use strong passwords and must not share passwords with anyone.

Passwords must not be stored in plain text, email, chat messages, spreadsheets, or documents.

## 5. Public Wi-Fi Usage
Employees must avoid accessing confidential systems on public Wi-Fi unless using company-approved VPN.

Public Wi-Fi should not be used for sensitive financial, customer, or production system activities unless secure VPN is active.

## 6. Data Storage Rules
Confidential company data and customer data must be stored only in approved company systems.

Employees must not store confidential data on personal drives, personal cloud storage, USB drives, or unmanaged devices.

## 7. Incident Reporting
Employees must report lost devices, suspected phishing, malware alerts, unauthorized access, or accidental data exposure to IT Security immediately.

## 8. Relationship to AI Usage Policy
If an employee wants to use AI tools with company data, the AI Usage Policy must also be followed.
